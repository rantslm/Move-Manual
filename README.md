# Mini-Project-1

This is the first major coding project designed to consolidate everything learned in Modules 1 - 4: HTML, CSS, JavaScript, Fetch, Axios, Bootstrap


Display workout plans or exercises fetched from API
Use cards for exercises with images and details
Allow filtering by muscle group or difficulty

PURPOSE 
to help users do three things: 1) discover 2) save 3) organize personalized workout exercises. it does this through taking exercise data from an API and allowing users to filter through categories like muscle group, difficulty, workout type, and equipment needed. 

PROBLEMS...SOLVED
-info overload…everything in one place instead of blogs, articles, youtube videos, tiktoks
-custom exercise plans without needing a trainer or a paid fitness subscription. 
-filters make a personalized workout plan. workouts are NOT one size fits all.


TARGET AUDIENCE 
adults 18-40 y.o. who want to stay active (at home or at gym) with simple tools to help plan exercises quickly, without needing a coach or subscription. 

USER STORIES
--ME…scoliosis, physical therapy, specific body parts
--gym bro--equipment connoisseur, workout types
--almond mom

POTENTIAL BUSINESS VALUE
reduced search time and increase user engagement
favorites system=habit-forming behavior
charts imply future features like progress tracking, recommended routines 
eventually can expand into trainer profiles, integration with wearables(smart watch), gym partnerships(a machine with a QR code to workouts specific to the equipment)
